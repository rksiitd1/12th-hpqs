Important Formulas (Class 12 Math)

समाकलन (Integration)
∫ x^n dx = x^{n+1}/(n+1)
∫ (1/x) dx = log|x|
∫ e^x dx = e^x
∫ a^x dx = a^x / log a
∫ tan x dx = log|sec x|
∫ cot x dx = log|sin x|
∫ sec x dx = log|sec x + tan x|
∫ (1/√(a²-x²)) = sin⁻¹(x/a)
∫ (1/(a²+x²)) = (1/a)tan⁻¹(x/a)
∫ (1/√(x²-a²)) = log|x + √(x²-a²)|
∫ √(a²-x²) dx = (x/2)√... + (a²/2)sin⁻¹(x/a)
निश्चित समाकलन (Properties)
∫₀ᵃ f(x) dx = ∫₀ᵃ f(a-x) dx
∫₋ₐᵃ f(x) dx = 0 (यदि विषम/odd)
∫₋ₐᵃ f(x) dx = 2∫₀ᵃ f(x) (यदि सम/even)
ILATE Rule (By Parts)
∫uv = u∫v - ∫(u'∫v)dx
अवकलन (Differentiation)
d/dx(x^n) = nx^{n-1}
d/dx(sin x) = cos x
d/dx(cos x) = -sin x
d/dx(log x) = 1/x
d/dx(a^x) = a^x log a
d/dx(sin⁻¹x) = 1/√(1-x²)
d/dx(tan⁻¹x) = 1/(1+x²)
d/dx(sec⁻¹x) = 1/(|x|√(x²-1))
अवकल समीकरण (Diff. Eq.)
1. Homogeneous: Put y = vx
2. Linear (LDE): dy/dx + Py = Q
I.F. = e^{∫ P dx}
Sol: y(I.F.) = ∫ Q(I.F.) dx + c
सदिश बीजगणित (Vectors)
Unit Vector: â = &vec;a / |&vec;a|
Dot: &vec;a ċ &vec;b = |a||b|cosθ
Cross: &vec;a × &vec;b = |a||b|sinθ n̂
Projection of &vec;a on &vec;b: (&vec;a ċ &vec;b) / |&vec;b|
Perpendicular: &vec;a ċ &vec;b = 0
Parallel: &vec;a × &vec;b = 0
Area of Δ: (1/2) |&vec;a × &vec;b|
त्रिविमीय ज्यामिति (3D Geo)
Direction Cosines (l, m, n):
l = cosα, m = cosβ, n = cosγ
l² + m² + n² = 1
Distance Formula:
√((x₂-x₁)² + (y₂-y₁)² + (z₂-z₁)²)
Angle between Lines:
cosθ = |a₁a₂ + b₁b₂ + c₁c₂| / √...√...
Shortest Distance (Skew Lines):
d = |(&vec;b₁ × &vec;b₂) ċ (&vec;a₂ - &vec;a₁)| / |&vec;b₁ × &vec;b₂|
Equation of Plane (Intercept):
x/a + y/b + z/c = 1
प्रतिलोम त्रिकोणमिति (ITF)
sin⁻¹x + cos⁻¹x = π/2
tan⁻¹x + cot⁻¹x = π/2
tan⁻¹x + tan⁻¹y = tan⁻¹((x+y)/(1-xy))
2tan⁻¹x = tan⁻¹((2x)/(1-x²))
2tan⁻¹x = sin⁻¹((2x)/(1+x²))
आव्यूह (Matrices)
(AB)' = B'A'
(AB)⁻¹ = B⁻¹A⁻¹
Inverse: A⁻¹ = (1/|A|) adj A
Area of Δ: (1/2) |det(x,y,1)|
Singular Matrix: |A| = 0
प्रायिकता (Probability)
P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
P(A|B) = P(A ∩ B) / P(B)
Independent Events:
P(A ∩ B) = P(A) ċ P(B)
Mean (Expectation):
μ = ∑ xᵢ P(xᵢ)
संबंध और फलन
Reflexive: (a,a) ∈ R
Symmetric: (a,b) ⇒ (b,a)
Transitive: (a,b),(b,c) ⇒ (a,c)
Fog: fog(x) = f(g(x))
