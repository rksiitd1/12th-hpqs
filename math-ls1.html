PART A: Long Answer Type (5 Marks)
Q1. Z = 5x + 10y का अधिकतमीकरण एवं न्यूनतमीकरण करें, जबकि x + 2y ≤ 120, x + y ≥ 60, x - 2y ≥ 0, x, y ≥ 0.
संकेत (Hint):
  1. असमिकाओं का ग्राफ खींचें और सुसंगत क्षेत्र (Feasible Region) छायांकित करें।
  2. कोणीय बिंदुओं (Corner Points) के निर्देशांक ज्ञात करें।
  3. Z = 5x + 10y में मान रखकर अधिकतम/न्यूनतम चुनें।
Q2. यदि tan⁻¹x + tan⁻¹y + tan⁻¹z = π/2, तो सिद्ध करें कि yz + zx + xy = 1.
संकेत (Hint): सूत्र tan⁻¹[(x+y+z - xyz)/(1 - xy - yz - zx)] = π/2 का प्रयोग करें। हर (Denominator) शून्य के बराबर होगा।
Q3. मान ज्ञात करें: ∫0π/2 log(sin x) dx.
संकेत (Hint): P4 प्रगुण (∫f(x) = ∫f(a-x)) का प्रयोग करें। I + I = 2I जोड़ने पर log(sin 2x) - log 2 प्राप्त होगा। उत्तर: -(π/2) log 2.
Q4. हल करें: (x² - y²) dy/dx = 2xy.
संकेत (Hint): यह समघातीय (Homogeneous) अवकल समीकरण है। y = vx रखें।
Q5. गुणनखंड निकालें:
(b+c)²a²a²
b²(c+a)²b²
c²c²(a+b)²
संकेत (Hint): C₁ → C₁ - C₃ और C₂ → C₂ - C₃ संक्रिया करें। उभयनिष्ठ बाहर निकालें।
Q6. dy/dx निकालें, जब xy + yx = 1.
संकेत (Hint): u = xy और v = yx मानकर log लेकर अवकलन करें। du/dx + dv/dx = 0.
Q7. तीन सिक्कों की उछाल में शीर्षों (Heads) की संख्या का माध्य ज्ञात करें।
संकेत (Hint): यादृच्छिक चर X = 0, 1, 2, 3. प्रायिकता P(X) निकालें और μ = ∑XP(X) का प्रयोग करें।
Q8. मान ज्ञात करें: (î - 3ĵ + 4k̂) ċ [(2î - ĵ) × (ĵ + k̂)].
संकेत (Hint): पहले क्रॉस गुणन (Cross Product) निकालें, फिर डॉट गुणन (Dot Product)।
Q9. आव्यूह विधि से हल करें: 2x + 3y + 3z = 5, x - 2y + z = -4, 3x - y - 2z = 3.
संकेत (Hint): AX = B रूप में लिखें। A⁻¹ = (adj A)/|A| निकालें। X = A⁻¹B.
Q10. हल करें: (1+x²) dy/dx + y = tan⁻¹x.
संकेत (Hint): रैखिक अवकल समीकरण (LDE)। I.F. = e^(tan⁻¹x).
Q11. रेखाओं r = (î+2ĵ+k̂) + λ(î-2ĵ+k̂) तथा r = (2î-2ĵ-k̂) + μ(î+ĵ+2k̂) के बीच की न्यूनतम दूरी ज्ञात करें।
संकेत (Hint): विषम-तलीय (Skew Lines) दूरी का सूत्र: d = |(b₁×b₂)·(a₂-a₁)| / |b₁×b₂|.
Q12. हल करें: ∫01 x log(1 + 2x) dx.
संकेत (Hint): खंडशः समाकलन (Integration by Parts) का प्रयोग करें (ILATE rule).
Q13. सिद्ध कीजिए (AB)⁻¹ = B⁻¹A⁻¹, जहाँ A और B 3x3 आव्यूह दिए गए हैं।
संकेत (Hint): LHS के लिए AB का गुणन कर Inverse निकालें। RHS के लिए अलग-अलग Inverse निकालें।
Q14. यदि (x + y) = sec⁻¹(x - y) तो dy/dx ज्ञात करें।
संकेत (Hint): sec(x+y) = x-y लिखकर अवकलन करें।
Q15. हल करें: (1 + tan y)(dx - dy) + 2x dy = 0.
संकेत (Hint): dx/dy रूप में बदलें और रैखिक अवकल समीकरण (LDE) की जाँच करें।
Q16. तल 3x + 4y - 5z = 2 के सापेक्ष बिन्दु P(3, 1, -4) का प्रतिबिम्ब ज्ञात कीजिए।
संकेत (Hint): सूत्र (x-x₁)/a = (y-y₁)/b = (z-z₁)/c = -2(ax₁+by₁+cz₁+d)/(a²+b²+c²) का प्रयोग करें।
Q17. समाकलन करें: ∫ tan⁴ x dx.
संकेत (Hint): tan²x · (sec²x - 1) के रूप में तोड़ें।
Q18. वक्र 3x² - y² = 8 पर उन अभिलम्ब रेखाओं का समीकरण ज्ञात करें जो रेखा x + 3y = 4 के समांतर हैं।
संकेत (Hint): अभिलम्ब की ढाल = -1/3। वक्र का अवकलन कर स्पर्श बिंदु निकालें।
Q19. दो पासों को चार बार फेंकने में कम से कम दो बार द्विक (doublets) आने की प्रायिकता क्या है?
संकेत (Hint): बरनौली परीक्षण (n=4, p=1/6). P(X≥2) = 1 - [P(0)+P(1)].
Q20. यदि a = î-ĵ+k̂ तथा b = ĵ-k̂, तो c ज्ञात करें ताकि a × c = b तथा a ċ c = 3.
