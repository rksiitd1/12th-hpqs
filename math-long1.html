Q1. Z = 5x + 10y का अधिकतमीकरण एवं न्यूनतमीकरण करें, जबकि x + 2y ≤ 120, x + y ≥ 60, x - 2y ≥ 0, x, y ≥ 0.
संकेत (Hint):
  1. दिए गए असमिकाओं (inequalities) को समीकरण मानकर ग्राफ खींचे।
  2. उभयनिष्ठ सुसंगत क्षेत्र (Feasible Region) को छायांकित करें।
  3. क्षेत्र के कोणीय बिंदुओं (Corner Points) के निर्देशांक ज्ञात करें।
  4. प्रत्येक बिंदु पर Z का मान निकालें। सबसे बड़ा मान 'अधिकतम' और सबसे छोटा 'न्यूनतम' होगा।
Q2. यदि tan⁻¹x + tan⁻¹y + tan⁻¹z = π/2, तो सिद्ध करें कि yz + zx + xy = 1.
संकेत (Hint): सूत्र का प्रयोग करें:
tan⁻¹[(x+y+z - xyz) / (1 - xy - yz - zx)] = π/2
चूँकि tan(π/2) = ∞ (या 1/0), अतः हर (denominator) को 0 के बराबर रखें:
1 - xy - yz - zx = 0 ⇒ yz + zx + xy = 1.
Q3. मान ज्ञात करें: ∫0π/2 log(sin x) dx.
संकेत (Hint):
  1. इसे I मानें। गुणधर्म ∫0a f(x)dx = ∫0a f(a-x)dx का प्रयोग करें।
  2. I = ∫ log(cos x) dx प्राप्त होगा। दोनों समीकरणों को जोड़ें (2I)।
  3. log(sin x) + log(cos x) = log(sin 2x) - log 2 का प्रयोग करें।
  4. अंतिम उत्तर: -(π/2) log 2
Q4. हल करें: (x² - y²) dy/dx = 2xy.
संकेत (Hint): यह एक समघातीय अवकल समीकरण (Homogeneous D.E.) है।
  • y = vx रखें, तो dy/dx = v + x(dv/dx) होगा।
  • मान रखकर चरों को अलग करें (Variable Separation)।
  • समाकलन करें: ∫[1/(1+v²)] dv आदि का प्रयोग होगा।
Q5. गुणनखंड निकालें:
(b+c)²a²a²
b²(c+a)²b²
c²c²(a+b)²
संकेत (Hint): संक्रियाओं (Operations) का प्रयोग कर अधिकतम शून्य बनायें:
  • C₁ → C₁ - C₃ और C₂ → C₂ - C₃ करें।
  • (a² - b²) = (a-b)(a+b) सूत्र का प्रयोग कर उभयनिष्ठ (Common) निकालें।
  • अंत में सारणिक का विस्तार करें। उत्तर: 2abc(a+b+c)³
Q6. dy/dx निकालें, जब xy + yx = 1.
संकेत (Hint): फलन की घात फलन है, अतः लघुगणक (Log) का प्रयोग करें।
  • u = xy तथा v = yx मानें, तो u + v = 1.
  • du/dx + dv/dx = 0.
  • अलग-अलग log लेकर अवकलन करें: log u = y log x.
Q7. तीन सिक्कों की उछाल में शीर्षों (Heads) की संख्या का माध्य ज्ञात करें।
संकेत (Hint):
  • यादृच्छिक चर X (शीर्षों की संख्या) के मान 0, 1, 2, 3 हो सकते हैं।
  • प्रायिकता P(X) निकालें: P(0)=1/8, P(1)=3/8...
  • माध्य (Mean) का सूत्र: μ = ∑ XċP(X)
Q8. मान ज्ञात करें: (î - 3ĵ + 4k̂) ċ [(2î - ĵ) × (ĵ + k̂)].
संकेत (Hint): यह अदिश त्रिक गुणन (Scalar Triple Product) है।
  1. पहले कोष्ठक [...] के अंदर सदिश गुणन (Cross Product) निकालें।
  2. प्राप्त सदिश का पहले सदिश के साथ बिंदु गुणन (Dot Product) करें।
  3. वैकल्पिक विधि: सारणिक (Determinant) में तीनों सदिशों के गुणांक रखकर मान निकालें।
Q9. आव्यूह विधि से हल करें: 2x + 3y + 3z = 5, x - 2y + z = -4, 3x - y - 2z = 3.
संकेत (Hint): समीकरण को AX = B रूप में लिखें।
  • |A| का मान निकालें (शून्य नहीं होना चाहिए)।
  • सहखंडज आव्यूह (adj A) निकालें।
  • A⁻¹ = (1/|A|) × (adj A) का प्रयोग करें।
  • अंत में X = A⁻¹B गुणा करके x, y, z निकालें।
Q10. हल करें: (1+x²) dy/dx + y = tan⁻¹x.
संकेत (Hint): यह रैखिक अवकल समीकरण (L.D.E.) है: dy/dx + Py = Q.
  • P = 1/(1+x²) पहचानें।
  • समाकलन गुणांक (I.F.) निकालें: I.F. = e∫ P dx = etan⁻¹x.
  • हल: y(I.F.) = ∫ Q(I.F.) dx + C.
Q11. रेखाओं r = (î+2ĵ+k̂) + λ(î-2ĵ+k̂) तथा r = (2î-2ĵ-k̂) + μ(î+ĵ+2k̂) के बीच की न्यूनतम दूरी ज्ञात करें।
संकेत (Hint): विषम-तलीय रेखाओं (Skew Lines) के बीच की दूरी का सूत्र:
d = | (b₁ × b₂) ċ (a₂ - a₁) | / |b₁ × b₂|
यहाँ b₁ और b₂ दिशा सदिश हैं। इनका क्रॉस गुणन निकालें।
Q12. हल करें: ∫01 x log(1 + 2x) dx.
संकेत (Hint): खंडशः समाकलन (Integration by Parts) का प्रयोग करें।
  • ILATE नियम के अनुसार: u = log(1+2x) (पहला फलन) और v = x (दूसरा फलन)।
  • सूत्र: ∫uv dx = u∫v dx - ∫[(du/dx)∫v dx] dx.
  • अंत में सीमाएँ (Limits) 0 से 1 रखें।
Q13. सिद्ध कीजिए (AB)⁻¹ = B⁻¹A⁻¹, जहाँ A और B 3x3 आव्यूह दिए गए हैं।
संकेत (Hint): यह प्रश्न गणना आधारित है।
  1. AB का गुणा करें और उसका व्युत्क्रम (Inverse) निकालें (LHS)।
  2. B का Inverse और A का Inverse अलग-अलग निकालें।
  3. B⁻¹A⁻¹ का गुणा करें (RHS) और दिखाएं कि दोनों समान हैं।
Q14. यदि (x + y) = sec⁻¹(x - y) तो dy/dx ज्ञात करें।
संकेत (Hint):
  • त्रिकोणमितीय रूपांतरण: sec(x + y) = x - y.
  • अब x के सापेक्ष दोनों पक्षों का अवकलन करें।
  • sec(x+y)tan(x+y) [1 + dy/dx] = 1 - dy/dx.
  • dy/dx वाले पदों को एक तरफ करके हल करें।
Q15. हल करें: (1 + tan y)(dx - dy) + 2x dy = 0.
संकेत (Hint): इस समीकरण को dy/dx या dx/dy के रूप में व्यवस्थित करें।
यह dx/dy + Px = Q (रैखिक) रूप में बदल सकता है।
I.F. निकालकर हल करें।
Q16. तल 3x + 4y - 5z = 2 के सापेक्ष बिन्दु P(3, 1, -4) का प्रतिबिम्ब ज्ञात कीजिए।
संकेत (Hint):
  • माना प्रतिबिम्ब Q(h, k, w) है।
  • सूत्र: (x-x₁)/a = (y-y₁)/b = (z-z₁)/c = -2(ax₁+by₁+cz₁+d) / (a²+b²+c²).
  • यहाँ a,b,c तल के अभिलंब (3, 4, -5) हैं। हल करके h, k, w निकालें।
Q17. समाकलन करें: ∫ tan⁴ x dx.
संकेत (Hint):
  • इसे ∫tan²x ċ tan²x dx लिखें।
  • एक tan²x को (sec²x - 1) में बदलें।
  • ∫(tan²x sec²x - tan²x) dx.
  • प्रथम भाग में tan x = t मानें, दूसरे भाग में tan²x को फिर (sec²x - 1) लिखें।
Q18. वक्र 3x² - y² = 8 पर उन अभिलम्ब रेखाओं का समीकरण ज्ञात करें जो रेखा x + 3y = 4 के समांतर हैं।
संकेत (Hint):
  1. दी गई रेखा की ढाल m = -1/3 है। चूँकि अभिलम्ब समांतर है, अभिलम्ब की ढाल भी -1/3 होगी।
  2. वक्र का अवकलन कर स्पर्श रेखा की ढाल (m_t) निकालें।
  3. अभिलम्ब की ढाल = -1 / m_t = -1/3 रखकर संपर्क बिंदु (x, y) निकालें।
  4. बिंदु-ढाल रूप से समीकरण लिखें।
Q19. दो पासों को चार बार फेंकने में कम से कम दो बार द्विक (doublets) आने की प्रायिकता क्या है?
संकेत (Hint): यह बरनौली परीक्षण (Bernoulli Trials) है।
  • n = 4. द्विक (जैसे 1,1; 2,2...) आने की प्रायिकता p = 6/36 = 1/6। q = 5/6।
  • "कम से कम दो बार" = P(X ≥ 2) = 1 - [P(0) + P(1)]।
  • सूत्र P(X=r) = nCr pr qn-r का प्रयोग करें।
Q20. यदि a = î-ĵ+k̂ तथा b = ĵ-k̂, तो c ज्ञात करें ताकि a × c = b तथा a ċ c = 3.
संकेत (Hint):
  • माना c = xî + yĵ + zk̂.
  • a ċ c = 3 ⇒ x - y + z = 3 (समीकरण 1).
  • a × c निकालें और उसे b के बराबर रखकर x, y, z में अन्य समीकरण प्राप्त करें।
  • समीकरणों को हल करके x, y, z निकालें।
