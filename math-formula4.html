4. अवकलज के अनुप्रयोग (AOD)

Slope of Tangent (m): dy/dx at (x₁,y₁)
Slope of Normal: -1 / m
Equation: y - y₁ = m(x - x₁)

Increasing: f'(x) > 0 | Decreasing: f'(x) < 0
Maxima/Minima (2nd Order Test):
1. Find f'(x)=0 to get critical points c.
2. Find f''(c).
If f''(c) < 0 ⇒ Maxima.
If f''(c) > 0 ⇒ Minima.

7. अवकल समीकरण (Diff. Eq.)

Variable Separable: ∫f(y)dy = ∫g(x)dx
Homogeneous: Put y = vx, then dy/dx = v + x(dv/dx)
Linear Diff Eq (LDE):
Form: dy/dx + Py = Q
I.F. (Integrating Factor) = e^∫P dx
Solution: y(I.F.) = ∫(Q × I.F.)dx + C
