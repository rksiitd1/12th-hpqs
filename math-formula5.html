2. आव्यूह (Matrices) & सारणिक (Determinants)

Types (प्रकार)

A = Aᵀ → Symmetric
A = -Aᵀ → Skew-Symmetric
Skew में diagonal = 0

Adjoint Properties

A(adj A) = (adj A)A = |A|I
|adj A| = |A|n-1 (order n)
adj(AB) = (adj B)(adj A)
adj(adj A) = |A|n-2A
adj(kA) = kn-1(adj A)

Inverse Properties

A⁻¹ = 1|A| (adj A)
(AB)⁻¹ = B⁻¹A⁻¹
(Aᵀ)⁻¹ = (A⁻¹)ᵀ
|A⁻¹| = 1|A|
(A⁻¹)⁻¹ = A

Determinant Properties

|AB| = |A||B|
|kA| = kn|A| (order n)
|Aᵀ| = |A|

Area of Triangle

Δ = 12 |x₁(y₂-y₃) + x₂(y₃-y₁) + x₃(y₁-y₂)|

Matrix Method (System of Equations)

AX = B ⇒ X = A⁻¹B (जब |A| ≠ 0)
Unique Solution: |A| ≠ 0
No Solution: |A| = 0, (adj A)B ≠ 0
Infinite Solutions: |A| = 0, (adj A)B = 0

Standard Derivatives

4. अवकलज के अनुप्रयोग (AOD)

Rate of Change

Instantaneous Rate = dydx

Tangent & Normal

Slope of Tangent = m = dydx|(x₁,y₁)
Slope of Normal = -1m
Tangent Equation: y-y₁ = m(x-x₁)
Normal Equation: y-y₁ = -1m(x-x₁)

Increasing/Decreasing

Increasing: f'(x) ≥ 0
Decreasing: f'(x) ≤ 0
Strictly Inc: f'(x) > 0
Strictly Dec: f'(x) < 0

Maxima & Minima

Critical Point: f'(x) = 0
First Derivative Test:
• Maxima: f' changes + to -
• Minima: f' changes - to +
Second Derivative Test:
• f''(c) > 0 → Minima at x=c
• f''(c) < 0 → Maxima at x=c
• f''(c) = 0 → Test fails
