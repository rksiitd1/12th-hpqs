1. सम्बन्ध एवं फलन (Relations & Functions)

• स्वतुल्य (Reflexive): (a,a) ∈ R, ∀ a ∈ A
• सममित (Symmetric): (a,b)∈R ⇒ (b,a)∈R
• संक्रामक (Transitive): (a,b)∈R, (b,c)∈R ⇒ (a,c)∈R
• Binary Operation: a∗b; Identity e: a∗e=a=e∗a

3. आव्यूह और सारणिक (Matrices & Det)

• सममित (Symmetric): A' = A
• विषम-सममित (Skew-Sym): A' = -A
Adjoint & Inverse (Most Imp):
1. A(adj A) = |A| I
2. |adj A| = |A|ⁿ⁻¹ (n = order)
3. (AB)⁻¹ = B⁻¹A⁻¹ | (A')⁻¹ = (A⁻¹)'
4. A⁻¹ = 1|A|(adj A) (if |A|≠0)
• Area of Δ: ½ |x₁(y₂-y₃) + x₂(y₃-y₁) + ...|
Using Det: Area = ½ |det|. Collinear if Area = 0.
• Linear Eq: AX = B ⇒ X = A⁻¹B

4. सांतत्य तथा अवकलनीयता (Cont & Diff)

• Continuity: LHL = RHL = f(a)
• Chain Rule: dy/dx = dy/dt × dt/dx
• Product: (uv)' = u'v + uv'
• Quotient: (u/v)' = u'v - uv'v²
d(xⁿ) = nxⁿ⁻¹ d(eˣ) = eˣ d(aˣ) = aˣln a
d(ln x) = 1/x d(√x) = 1/2√x d(1/x) = -1/x²
Trig Derivatives:
tan x → sec²x | cot x → -cosec²x
sec x → sec x tan x | cosec x → -cosec x cot x
Inverse Trig Diff:
sin⁻¹x → 1√(1-x²)   |   tan⁻¹x → 11+x²
sec⁻¹x → 1|x|√(x²-1)
Log Diff: If y=xˣ, dy/dx = xˣ(1+ln x)
Parametric: dy/dx = (dy/dθ) / (dx/dθ)

5. अवकलज के अनुप्रयोग (AOD)

• Rate Measure: dy/dt
• Tangent Slope (m): dy/dx at point.
• Normal Slope: -1/m
• Eq: y-y₁ = m(x-x₁)
• Approximation: f(x+Δx) ≈ f(x) + f'(x)Δx
Maxima/Minima (महत्तम/न्यूनतम):
1. f'(x)=0 find critical pts.
2. Check f''(x).
• f''(x) < 0 ⇒ Maxima (महत्तम)
• f''(x) > 0 ⇒ Minima (न्यूनतम)
• Increasing: f'(x) > 0 | Decreasing: f'(x) < 0

8. अवकल समीकरण (Diff Eq.)

9. सदिश बीजगणित (Vector Algebra)

• |a| = √(x²+y²+z²)
• Unit: a = a / |a|
• Dir Cosines: l,m,n where l²+m²+n²=1
Dot Product (अदिश गुणन):
a⋅b = |a||b|cosθ
• If a⊥b ⇒ a⋅b = 0
• Projection (a on b): (a⋅b) / |b|
Cross Product (सदिश गुणन):
a×b = |a||b|sinθ n
• If a||b ⇒ a×b = 0
• Area (Δ): ½|a×b| | Area (||gm): |a×b|

10. त्रिविमीय ज्यामिति (3D Geometry)

Direction Cosines: l=a/√(a²+b²+c²) etc.
Line (रेखा):
Vector: r = a + λb
Cartesian: x-x₁a = y-y₁b = z-z₁c
Plane (समतल):
Vector: r⋅n = d (Normal form)
Cartesian: Ax + By + Cz + D = 0
Intercept: x/a + y/b + z/c = 1
Shortest Distance (Skew Lines):
d = |(b₁×b₂) ⋅ (a₂-a₁)||b₁×b₂|
Angle: cosθ = |a⋅b||a||b| (for lines use b vector, for planes use n vector).

11. प्रायिकता (Probability)

• P(A∪B) = P(A) + P(B) - P(A∩B)
• Conditional: P(A|B) = P(A∩B) / P(B)
• Independent: P(A∩B) = P(A)⋅P(B)
Bayes' Theorem:
P(E₁|A) = P(E₁)P(A|E₁)∑ P(Eᵢ)P(A|Eᵢ)
Distribution (बंटन):
Mean (μ) = ∑xᵢpᵢ | Var (σ²) = ∑xᵢ²pᵢ - (μ)²
